# Curated Learning Resources

> Free & open-source prioritized. Paid resources marked with 💰.

## Python Core

- 📖 [Fluent Python, 2nd Edition](https://www.oreilly.com/library/view/fluent-python-2nd/9781492056348/) — Luciano Ramalho (💰, the Python bible)
- 📖 [Python Official Docs](https://docs.python.org/3/) — best reference, always up-to-date
- 🎓 [CS50P (Harvard)](https://cs50.harvard.edu/python/) — free, excellent intro course
- 📖 [Real Python](https://realpython.com/) — deep-dive tutorials
- 📖 [Python 3 Patterns, Recipes and Idioms](https://python-3-patterns-idioms-test.readthedocs.io/) — free
- 📺 [mCoding (YouTube)](https://www.youtube.com/@mCoding) — advanced Python deep dives
- 📖 [CPython Internals](https://realpython.com/products/cpython-internals-book/) — understand the interpreter

## DSA

- 📖 [Problem Solving with Algorithms (Python)](https://runestone.academy/ns/books/published/pythonds/index.html) — free interactive book
- 📖 [Competitive Programming 3](https://cpbook.net/) — Halim & Halim (💰)
- 🎓 [MIT 6.006 Introduction to Algorithms](https://ocw.mit.edu/courses/6-006-introduction-to-algorithms-spring-2020/) — free OCW
- 🎓 [MIT 6.046 Design & Analysis of Algorithms](https://ocw.mit.edu/courses/6-046j-design-and-analysis-of-algorithms-spring-2015/) — advanced
- 📖 [Algorithms (Sedgewick & Wayne)](https://algs4.cs.princeton.edu/) — Java-based but theory is universal
- 💻 [LeetCode](https://leetcode.com/) / [NeetCode](https://neetcode.io/) — structured problem sets
- 📺 [Abdul Bari (YouTube)](https://www.youtube.com/@abdul_bari) — best DSA explanations

## OOP & Design Patterns

- 📖 [Architecture Patterns with Python](https://www.cosmicpython.com/) — free, DDD + patterns + Python
- 📖 [Refactoring.Guru](https://refactoring.guru/design-patterns) — patterns with Python examples
- 📺 [ArjanCodes (YouTube)](https://www.youtube.com/@ArjanCodes) — practical Python design
- 📖 [Python Design Patterns](https://github.com/faif/python-patterns) — opensource collection
- 📖 [Design Patterns (GoF)](https://en.wikipedia.org/wiki/Design_Patterns) — the original (💰)
- 🎓 [Design Patterns in Python (Udemy)](https://www.udemy.com/course/design-patterns-python/) — (💰)

## AI & DL Libraries

- 📖 [Python Data Science Handbook](https://jakevdp.github.io/PythonDataScienceHandbook/) — VanderPlas (free)

## Recommended Learning Path by Resource Intensity

| Budget | Path |
|--------|------|
| **Free only** | CS50P → Runestone DSA → Cosmic Python → VanderPlas Handbook → fast.ai → HuggingFace Course → CS231n notes → ISLR |
| **Minimal budget** | Above + Fluent Python ($50) + CP3 ($30) |
| **Full budget** | Above + O'Reilly subscription for all books + Coursera/DeepLearning.AI specialization |
