## Key Math

- Matrix multiply: $C_{ij} = \sum_k A_{ik} B_{kj}$
- Backprop: $\frac{\partial L}{\partial w} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial z} \cdot \frac{\partial z}{\partial w}$
- Gradient descent: $w_{t+1} = w_t - \eta \nabla L(w_t)$
- F1: $2 \cdot \frac{P \cdot R}{P + R}$
