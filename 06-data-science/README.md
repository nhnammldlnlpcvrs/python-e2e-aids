# Stage 6: Data Science & Analysis

End-to-end data science: SQL → Python → Statistics → Dashboard.

## Topics

1. Descriptive Statistics — moments, IQR, distributions
2. Probability Distributions — Bernoulli, Binomial, Poisson, Normal, Exponential
3. Inferential Statistics — CLT, confidence intervals, bootstrap
4. Hypothesis Testing — t-test, chi-square, ANOVA, p-values
5. Correlation & Regression — Pearson r, $R^2$, Ridge, Lasso
6. Bayesian Statistics — Bayes theorem, MCMC
7. SQL for Analytics — joins, window functions, CTEs
8. EDA Framework — univariate/bivariate/multivariate analysis
9. Visualization — Seaborn, Plotly, Streamlit
10. A/B Testing — sample size, power analysis, Cohen's d

## Key Math

- Normal PDF: $\mathcal{N}(x|\mu,\sigma^2) = \frac{1}{\sigma\sqrt{2\pi}} \exp(-\frac{(x-\mu)^2}{2\sigma^2})$
- CLT: $\bar{X}_n \sim \mathcal{N}(\mu, \sigma^2/n)$
- Pearson r, $R^2$, Bayes theorem, p-value definition
