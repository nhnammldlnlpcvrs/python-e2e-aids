# Stage 5: Domain Specialization

NLP & Computer Vision — from fundamentals to SOTA.

## NLP

1. Text Processing — tokenization (BPE), stemming, regex
2. Vector Space Models — BoW, TF-IDF, cosine similarity
3. Word Embeddings — Word2Vec (CBOW/Skip-gram), GloVe
4. Sequence Models — RNN, LSTM, GRU (all gates)
5. Attention & Transformers — self-attention, multi-head, positional encoding
6. Pretrained LMs — BERT, GPT, T5, fine-tuning
7. PEFT — LoRA, QLoRA
8. LLM Applications — RAG, agents, LangChain

## CV

1. Image Fundamentals — pixels, color spaces, histograms
2. Filtering — convolution, edge detection, morphological ops
3. CNN Architectures — AlexNet → ResNet → EfficientNet
4. Object Detection — YOLO, Faster R-CNN, NMS
5. Segmentation — U-Net, Mask R-CNN
6. Generative Models — VAE, GAN, Diffusion (DDPM)
7. Vision Transformers — ViT, patch embedding

**NLP:** TF-IDF, Cosine similarity, Attention(Q,K,V), LSTM gates, Cross-entropy
**CV:** 2D Convolution, IoU, Dice loss, GAN minmax objective
