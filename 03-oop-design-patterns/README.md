# Stage 3: OOP & Design Patterns

Design extensible, testable Python systems.

## Topics

1. Encapsulation — `@property`, `__slots__`, descriptor protocol
2. Inheritance — `super()`, MRO (C3 linearization), mixins
3. Polymorphism — duck typing, ABC, `Protocol` (structural subtyping)
4. SOLID Principles — S, O, L, I, D with Python examples
5. Creational Patterns — Singleton, Factory, Builder, Prototype
6. Structural Patterns — Adapter, Decorator, Facade, Proxy
7. Behavioral Patterns — Observer, Strategy, Command, State
8. Modern Python OOP — `dataclasses`, `Enum`, `TypeVar`, `Generic`
